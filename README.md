## Silo

Local-first AI "Chief of Staff" desktop app.

### Repo layout

- `apps/mcp-server`: MCP server (Rust, stdio JSON-RPC, LanceDB)
- `apps/desktop-ui`: Tauri desktop app (to be added)
- `crates/`: shared Rust crates (to be added)

### Dev

- Build/run MCP server:

```bash
cd apps/mcp-server
cargo run
```
